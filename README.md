Event Registration System

Overview

This project is a simple backend Event Registration System developed using Flask and SQLite. It allows users to view events and register for them. The project demonstrates backend development concepts such as API routes, database integration, and data management.

Features

- Create and manage events
- View available events
- Register users for events
- Store data using SQLite database
- Backend APIs built with Flask

Technologies Used

- Python
- Flask
- SQLite

Project Structure

- app.py
- event.db
- README.md
- requirements.txt

How to Run

1. Install Flask
2. Run the app.py file
3. Open the Flask server
4. Access the available routes

Learning Outcomes

- Backend development with Flask
- Database management using SQLite
- Creating API endpoints
- Handling user registrations and event data
